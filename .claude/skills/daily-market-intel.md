1. **AUTOMATED MODE**: You are running in an automated context. News data has been pre-fetched for you.
2. **CRITICAL**: You MUST use ONLY the URLs and articles from the provided NewsAPI JSON data. DO NOT make up or invent any URLs or articles.
3. Analyze the provided NewsAPI JSON data from the last 24 hours
4. Extract articles from the "articles" array within each competitor's data
